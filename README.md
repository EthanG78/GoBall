# GO BALL
An app that keeps you up to date with all of your favorite NBA all star's stats.

---
### Features

- Player search utility
- Player headshots
- 7+ stats for each player

---
### Statistics 
- Points Per Game
- Assists Per Game
- Rebounds Per Game
- Blocks Per Game
- Steals Per Game
- Three Point Percentage
- Free Throw Percentage
- Fieldm Goal Percentage

---
